# Timer-Capsule-Messenger
A web application that allows users to compose messages and schedule them to be delivered at a future date and time.

Time Capsule Messenger is a web-based application that allows users to write messages and schedule them to be sent or revealed in the future. It works like a digital time capsule, preserving thoughts, memories, or reminders until the chosen date and time.

## 🚀 Features

- Create and schedule future messages
- Select custom delivery date and time
- User-friendly and responsive interface
- Store messages securely
- View scheduled messages
- Clean and modern UI

## 🛠️ Tech Stack

- HTML
- CSS
- Bootstrap
- React.js
- JavaScript
